# Merchant POS Payment Prototype

A clickable POS payment prototype built with React + Vite.

This project simulates a merchant payment terminal with:

- Debit Card Payment Simulation
- QR Code Payment
- Transaction History (Stored in localStorage)
- Success / Failure Payment Simulation
- Card Number Generator
- Clean Merchant-Friendly UI
